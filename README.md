# Product Name

> Find and Delete duplicate files.


## Installation

OS X & Linux & Windows:

```sh
pip install git+git://github.com/andreztz/finddups.git
```

## Usage example

```sh
$ finddups --help
$ finddups --directory ~/Music
$ finddups --directory ~/Music --file-type mp3 --hash-type 
```
